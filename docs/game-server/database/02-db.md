## SQL 이란?

* RDBMS를 조작하기 위한 언어(명령어)
* MS-SQL은 T-SQL이라고 부르기도한다.(SQL에서 몇가지 기능이 추가 되었기에)
* 단, 표준 SQL을 쓰는 것을 추천...
* CRUD(Create / Read / Update / Delete)를 주로 다룬다.
