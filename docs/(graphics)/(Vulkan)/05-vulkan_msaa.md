## 문제: Aliasing (계단 현상) !!

* 컴퓨터 화면은 정사각형 픽셀로 이루어져 있다. 
* 그래서 대각선이나 곡선을 그리면 "계단처럼" 울퉁불퉁하게 보인다.

```
이상적인 선       실제 화면에서 보이는 모습
    /             ■
   /              ■■
  /                 ■■
 /                    ■■
                        ■  ← 계단 현상!
```

* 이걸 어떻게 자연스럽게 만들까?

---

## 해결책: 안티앨리어싱

* 가장자리를 부드럽게 만들어서 계단 현상을 줄이는 기술

```
// 4x MSAA - 픽셀 안의 4개 점을 검사
┌─────┐
│ ● ● │
│     │  ← 4개 중 몇 개가 삼각형 안에 있나?
│ ● ● │     → 비율에 따라 색상 블렌딩
└─────┘
```

```
MSAA 없음                    4x MSAA 적용
■■■                          ■■■░
  ■■■          →               ░■■░
    ■■■                          ░■■░
      ■                            ░■

(딱딱한 계단)                (부드러운 가장자리)
```

| 방식 | 결과 |
|:---|:---|
| MSAA 없음 | 픽셀이 완전히 켜지거나 꺼짐 (뚝뚝 끊김) |
| 4x MSAA | 4개 중 2개가 안에 있음 → 50% 투명도로 렌더링 |

| 장점 | 단점 |
|:---|:---|
| 가장자리가 부드러워짐 | 메모리 사용량 증가 (4x = 4배) |
| 구현이 비교적 간단 | GPU 성능 부하 |
| 하드웨어 가속 지원 | 텍스처 내부 앨리어싱은 해결 못함 |
