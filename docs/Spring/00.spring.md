## 자주사용되는 단축키

* Settings - Ctrl + Alt + s
* generate(constructor, getter, setter) - alt + insert
* go to define - ctrl + alt + b
* navigate - ctrl + n
* Unit Test - ctrl + shift + t
* for문 자동생성 - 'iter' + tab
* Override - Ctrl + o
* 서버 재시작 - Ctrl + F5
* 이전 포커스 이동 - ctrl + alt + 좌/우
