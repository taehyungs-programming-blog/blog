## Vue 인스턴스 선언

```html
<!-- ... -->

  <body>
    <div id="app">
      {{ message }}
    </div>

    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
    <script>
        // new를 하는 순간 Vue인스턴스가 생성된다.
      new Vue({
        el: '#app',
        data: {
          message: 'Hello Vue.js'
        }
      })
    </script>

<!-- ... -->
```
