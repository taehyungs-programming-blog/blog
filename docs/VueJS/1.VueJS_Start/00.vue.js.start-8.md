🐳 템플릿 문법은 크게 **두 가지**로 나뉜다.<br>
🐳 **데이터 바인딩**

```html
<div>{{ message }}</div>

<script>
    new Vue({
        data: {
            message: "This is data binding"
        }
    })
</script>
```

🐳 **디렉티브**

```html
<div>
    Hello <span v-if="show">Vue.JS</span>
</div>

<script>
    new Vue({
        data: {
            show: false
        }
    })
</script>
```

## Example - watch 써보기

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
</head>
<body>
  <div id="app">
    {{ num }}
    <button v-on:click="addNum">increase</button>
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        num: 10
      },
      // 데이터(num)의 변화에 따라 콘솔을찍게 하겠다.
      watch: {
        num: function() {
          this.logText();
        }
      },
      methods: {
        addNum: function() {
          this.num = this.num + 1;
        },
        logText: function() {
          console.log('changed');
        }
      }
    })
  </script>
</body>
</html>
```

## Example - Watch와 Computed 차이점은?

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
</head>
<body>
  <div id="app">
    {{ num }}
    {{ doubleNum }}
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        num: 10
      },
      computed: {   // 간단작업은 computed
        doubleNum: function() {
          return this.num * 2;
        }
      },
      watch: {      // 복잡한 로직의 작업은 watch
        num: function(newValue, oldValue) {
          this.fetchUserByNumber(newValue);
        }
      },
      methods: {
        fetchUserByNumber: function(num) {
          // console.log(num);
          axios.get(num);       // 이런식으로 데이터를 요청할 필요가 있을경우 watch활용
        }
      }
    });
  </script>
</body>
</html>
```

* [watch, computed 차이 공식문서 🌎](https://v2.vuejs.org/v2/guide/computed.html?redirect=true#ad)

## Example - 조건에 따라 class이름을 어떻게 넣을지 정할수 있음

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
  .warning {
    color: red;
  }
  </style>
</head>
<body>
  <div id="app">
    <p v-bind:class="errorTextColor">Hello</p>
  </div>
  
  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: {
        // cname: 'blue-text',
        isError: false
      },
      computed: {
        errorTextColor: function() {
          // if (isError) {
          //   return 'warning'
          // } else {
          //   return null;
          // }
          // 조건에 따라 class이름을 어떻게 넣을지 정할수 있음
          return this.isError ? 'warning' : null;
        }
      }
    });
  </script>
</body>
</html>
```
