* 레지스트리를 설정하자

```
HKEY_LOCAL_MACHINE
    SOFTWARE
        Microsoft
            Windows NT
                CurrentVersion
                    AeDebug
                        Debugger = "C:\Windows\system32\vsjitdebugger.exe" -p %ld -e %ld
                            Auto = 1
```

int wmain(int arcs, WCHAR *argv[]) {
