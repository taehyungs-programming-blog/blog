😺 `int`, `unsigned int`를 섞어서 사용하면 안된다.

😺 C++20에서 `cmp_XX`를 지원하며 비교를 가능하게 해준다

```cpp
#include <numbers>

// ...

cout << " -2 < 10 = " << cmp_less(x, y) << endl;
cout << " -2 <= 10 = " << cmp_less_equal(x, y) << endl;
cout << " -2 > 10 = " << cmp_greater(x, y) << endl;
cout << " -2 >= 10 = " << cmp_greater_equal(x, y) << endl;
```
