## 시작하기

* [Dioxus Docs ✨](https://dioxuslabs.com/learn/0.5/getting_started)
* 사전사항 : Rust

```shell
# dioxus 설치
$ cargo install dioxus-cli --locked

# 프로젝트 생성
$ dx new

# Web 시작
$ dx serve --hot-reload
```
