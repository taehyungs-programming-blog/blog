🛵 우선 현재 설치된 **vue-cli**의 버전확인

```bash
$ vue.cmd --version

# 만약 버전이 3.X 이하라면
$ npm install -g @vue/cli
```

🛵 프로젝트 생성 / 생성된 프로젝트 실행

```bash
$ vue.cmd create vue-todo
$ cd vue-todo
$ npm run serve
```
