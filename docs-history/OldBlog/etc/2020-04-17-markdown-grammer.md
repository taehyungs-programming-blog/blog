## (markdown) 링크 찾아가기

```s
[네임](#링크명)
```

위와 같이 링크를 걸면 링크로 들어가는데<br>
현재 io에서 링크명을 대문자 영어로 넣으면 링크를 못 찾아감<br>

아래와 같이 넣을 것!<br>

```s
[네임](#링크명(소문자))
```
