## Vue CLI 설치

```s
$ npm install -g @vue/cli
# vue-cli global 설치
$ vue --version
# 단순 버전확인
```

---

## Vue CLI 프로젝트 생성

```s
$ vue create vue-todo
# default 선택
$ cd vue-todo
$ npm run serve
```
